Hello, DWWW world

Project description

DWWW is an effort to build true distributed WWW infrastructures on top of existing WWW tools and resources: x?html, DOM (xHR, ..), Javascript.

Project's main page:
http://allter.github.com/dwww/.
Project's github page:
https://github.com/allter/dwww/.

Demos

  • 2 agents with simple content
  • Shows dns queries
  • Shows DWWW dns-based content resolver
  • Shows trust-based DDNS database propagation

Current plans:

  • Async messaging and ability to simulate timeouts
  • DDNS implementation

Distant iterations:

  • Proper HTML, DOM, CSS, Javascript virtualization (at least for Web1.0 DHTML applications)
  • Support for Opera Unite serverside
  • Support for Node.js serverside
  • Standalone DWWW-proxy and DDNS resolver tools
  • Cost sharing/rating management code
